# Security Policy

## Supported Versions

The latest released version is supported with security fixes.

| Version | Supported |
| ------- | --------- |
| 1.x     | Yes       |
| < 1.0   | No        |

## Reporting a Vulnerability

Please do not report security vulnerabilities in public issues.

Report vulnerabilities by creating a private security advisory in GitHub, or contact the maintainer through GitHub:

- https://github.com/SKonovalovS

Please include:

- affected version;
- Redmine version;
- steps to reproduce;
- impact description;
- suggested fix, if available.

## Scope

This plugin stores links and metadata for external sources. It does not synchronize external systems and does not require external API credentials.
